## 🚀 Quick Start

1. Clone the repository:

```bash
git clone https://github.com/nick33r/test-for-modul.git
```

2. Install dependencies and run dev server:

```bash
cd test-for-modul
npm install
npm run dev
```

3. For production build:

```bash
npm run build
```
